# 🍽️ Restaurant Management System

A responsive and user-friendly **Restaurant Management Web Application** that allows users to browse food items, add them to a cart, and simulate a checkout process. This project demonstrates real-world frontend development with dynamic interactions.

## 📌 Features

- 🍔 **Food Menu Display**
  - View different food items with price and details

- 🛒 **Add to Cart**
  - Add/remove items dynamically
  - Real-time cart updates

- 💰 **Total Price Calculation**
  - Automatically calculates total cost

- 💳 **Checkout System**
  - Simulated order placement

- 👤 **User Authentication Pages**
  - Login & Register UI

- 🛠️ **Admin Panel**
  - Manage food items (UI-based)

- 📱 **Responsive Design**
  - Works on mobile, tablet, and desktop

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript (Vanilla JS)** – Functionality & DOM manipulation
